## üîÑ State Update Patterns

### Adding a new item

```javascript
// Old state
state.items = [item1, item2]

// Action dispatched
dispatch(addItem(newItem))

// Reducer (immutable update)
state.items = [...state.items, newItem]

// New state
state.items = [item1, item2, newItem]
```

### Updating an item

```javascript
// Old state
state.items = [
  { id: 1, name: 'A' },
  { id: 2, name: 'B' }
]

// Action dispatched
dispatch(updateItem({ id: 1, name: 'Updated A' }))

// Reducer
const index = state.items.findIndex(i => i.id === action.payload.id)
state.items[index] = action.payload

// New state
state.items = [
  { id: 1, name: 'Updated A' },
  { id: 2, name: 'B' }
]
```

### Deleting an item

```javascript
// Old state
state.items = [item1, item2, item3]

// Action dispatched
dispatch(deleteItem(item2.id))

// Reducer
state.items = state.items.filter(i => i.id !== action.payload)

// New state
state.items = [item1, item3]
```
